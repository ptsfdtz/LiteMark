# 开发指南

## 1. 环境准备

1. Node.js 20+
2. pnpm 10+
3. Rust stable（桌面模式）

## 2. 安装依赖

```bash
pnpm install
```

## 3. 本地运行

```bash
pnpm dev
```

Tauri 桌面模式：

```bash
pnpm tauri dev
```

## 4. 常用命令

```bash
pnpm lint
pnpm build
pnpm preview
```

## 5. 当前阶段开发优先级

1. 先实现 Markdown MVP 主链路（编辑 + 预览 + 保存）。
2. 再补安全、模式一致性和测试。
3. 最后做性能与 Rust 下沉。

## 6. 提交流程建议

1. 小步提交，每个提交可回滚。
2. 提交前至少运行 `lint + build`。
3. 大改动需要同时更新 `docs/*`。
