4. Rust 能力缺口：Tauri 后端无业务命令，无法承载高性能下沉。
5. 测试缺口：无单元测试、无集成测试、无性能基准。
6. 文档缺口：README/CONTRIBUTING/CHANGELOG 已被重置或缺失。

8. 状态管理缺口：未建立统一状态策略（本地状态、持久化状态、后端状态）。
