# FAQ

## 1. 为什么要迁移到 shadcn/ui？

为了统一交互、降低重复样式维护成本，并提升可测试性。

## 2. 为什么不是一次性重写？

一次性重写回归风险高且难回滚。本项目采用按阶段可验收策略。

## 3. Markdown 为什么优先处理安全？

当前链路存在原始 HTML 渲染能力，若无严格 sanitize 容易产生 XSS 风险。

## 4. Rust 下沉会不会增加复杂度？

会增加一定复杂度，因此仅下沉 CPU 密集型路径，并保留 TS fallback。

## 5. 新人如何快速启动？

按 README 的快速启动步骤执行：安装依赖 -> `pnpm dev` -> `pnpm tauri dev`。
